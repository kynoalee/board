- 무대 타임 테이블 과 파일 변경법

1. flipboard 폴더에 들어간다.
2. input.html을 크롬으로 실행한다.
3. 시간과 한글, 영어 명을 기입힌다.
4. 시간같은 경우 시간 순서에 따라 작성해야한다.
5. 사용할 포스터 이미지 파일을 미리 flipboard/images 폴더에 넣어놓는다.
6. 해당 파일들을 ,로 구분해 기입한다.
7. 이미지 롤링의 순서는 기입한 순서대로이다.
ex) test.jpg 와 menu.png를 넣었다면 test.jpng,menu.png으로 기입한다.
8. 변경 버튼을 누르고 플립보드가 있는 브라우저를 클릭후, ctrl+R 이나 f5를 눌러 새로고침한다.
